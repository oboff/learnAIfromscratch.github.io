About this site

What is this

This is just a bunch of (draft) puzzles to figure out, I adjusted it so you can work F/T and complete it as a hobby by gamifying the material a little. I routinely eliminate material to increase the efficiency of the curriculum by hopefully reducing it to entirely problem sets. The concept is you can complete all the courses using a mobile device, which is what I do while on lunch breaks or commuting, sitting on the beach, or waiting in a line.

What do professors want you to learn

Transfer skills. Topic A is seemingly unrelated to topic B but they actually are related. Traditional curriculums struggle to achieve this as per Prof K's papers on teaching.

CompSci

Learn the foundations of computation itself, the trunk and branches.

  • CS19 and the book Programming and Programming Languages was chosen for having low complexity and ability to complete on a mobile device.
  • Robert Harpers PFPL book teaches the foundations of programming languages.
  • Ryan O'Donnell's Theoretical CS Toolkit course goes well with the book The Nature of Computation that explains the foundations of computation.
  • Competitive Programming chosen for practice. You can write text source on a tablet or phone and upload it to the online judge servers for free which compiles and tests them as many times as you want. Some of the languages are amenable to templating so often you are just writing the body of the function.

Math

  • Tao's book Analysis I and Wildberger's lectures reduce complexity.
  • The Putnam seminar by Prof Loh and the book Putnam and Beyond because competition problems teach you multiple things at the same time
  • Gilbert Strang's CSE book and MIT lectures to see the common applied framework for using math in all sciences, in a way it's the foundations of applied math

AI

Jump into the leaves. Google Colab can be used with a phone, so since we can practice writing AI algorithms on GPUs through Colab for free may as well also learn optimization of these algorithms. There are numerous AI competitions, most will accept notebook entries you can upload from colab.

Software

This site is built using org-mode which is a major mode for emacs meaning a collection of configurations that are automatically run whenever you open a .org file, a text file with some markup. I can edit the text on a phone then by a simple script ssh into a box to run: 

emacs myfile.org --batch -f org-html-export-to-html --kill

Which then is upload via version control, now with a minor mode to use persistent div id's instead of org-mode's randomly generated ones, so diffs and direct links are useful. The css background color choice was from a paper in readability, warmer colors are easier for us to read and their compliments at the other end of the color spectrum mix better. In org-mode you can disable the built-in css and set any css you want manually at the top of the file with style tags: 

#+HTML_HEAD: <style> body {background-color: #fafad2} </style>

The box that runs emacs is a guix system VM which is trivial to deploy you essentially edit some s-exp text which are settings inside lists, and it is a single declaration file how exactly you want your remote system set up. Invoking guix deploy installs the remote VM for you on digital ocean or elsewhere. Hacking around your own VPS is possible because the init system, packager, package builds, the initial RAM disk, system tests, and more are all written in a scheme variant which permits any other language you want to compile to their Guile virtual machine bytecode so you don't have to use their preferred language to manipulate your system you can use whatever you want.

To learn to manipulate your Guix system see this book (or pdf) by MIT, they reduce the complexity of learning what an operating system is to the essential details by studying a variant of the V6 original unix system. It is meant to be read while looking at the code.

Guix-HPC is used in high performance computing by INRIA researchers.

Guix-Jupyter notebooks

Guix-Jupyter is self-contained notebooks that describe their execution environment then can be deployed in a completely reproducible way, meaning you can freely install any software you want instead of being limited to what google colab offers and run your own VM hosted notebooks to do the AI curriculum on a mobile device if you wanted.
